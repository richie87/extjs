Infinite Scrolling

Ext JS's grid supports infinite scrolling, which enables you to load any number of records into a grid without paging.

The grid uses a virtualized scrolling system to handle potentially infinite data sets without any impact on client side performance.

The "Topic" column is configured wrap: true and allows text to wrap. This causes the buffered renderer to use variableRowHeight: true.

The number of rendered rows is only just larger than the visible row count. As the edge of the rendered data scrolls towards being in view, the table is refreshed, and repositioned to maintain visual scroll position.

The code is not minified, see infinite-scroll.js
